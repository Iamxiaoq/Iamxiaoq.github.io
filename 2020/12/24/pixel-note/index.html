刷twrp

参考
twrp下载

准备

需要先输入临时的twrp，然后通过临时的twrp刷入永久的

需提前准备以下三个文件:

  • twrp-3.1.1-0-fastboot-sailfish.img .img为临时镜像
  • twrp-pixel-installer-sailfish-3.1.1-0.zip .zip为永久镜像
  • VerifiedBootSigner-v3.zip patch boot img

推送两个zip包，后续通过临时的twrp刷入

adb push twrp-pixel-installer-sailfish-3.1.1-0.zip //sdcard/
adb push VerifiedBootSigner-v3.zip //sdcard/

输入临时twrp

adb reboot bootloader
fastboot boot twrp-3.1.1-0-fastboot-sailfish.img

刷入永久twrp

手机重启至临时twrp后依次刷入两个zip包
twrp gui install twrp-pixel-installer-sailfish-3.1.1-0.zip
twrp gui install VerifiedBootSigner-v3.zip

挂载/system为可写
